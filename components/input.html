# 验证提示

不能少于 6 个字
<d-input value="Vue" error="不能少于 6 个字"></d-input>

我们可以与 input 或者 blur 等事件相结合来实现动态提示：

不能少于 6 个字
<template>
  <d-input
    v-model="message"
    @input="validate"
    :error="error"
  ></d-input>
</template>
<script>
export default {
  data() {
    return {
      message: "vue",
      error: "不能少于 6 个字"
    };
  },
  methods: {
    validate(val) {
      if (val.length > 6) {
        this.error = "";
      } else {
        this.error = "不能少于 6 个字";
      }
    }
  }
};
</script>

# API

props：

名称 类型 描述
value String 输入框的值
disabled Boolean 是否禁用输入框
readonly Boolean 输入框是否只读
error String 显示的错误提示信息

事件：

名称 描述
change 值改变时触发
input 输入时触发
focus 输入框获得聚焦时触发
blur 输入框失焦时触发
