# 图标

本 UI 组件库带有默认的字体图标，通过 svg 来展示字体图标，并封装了 Icon 组件。

# 使用

默认的图标：

<d-icon name="settings"></d-icon>

图标是一个 svg 元素，具有默认类名 icon，你也可以给图标设置特定的类名： 

<d-icon class="personalClass" name="settings"></d-icon>

通过类名可以控制其大小等

.personalClass {
  width: 40px;
  height: 40px;
}
/* 由于内部使用 1em 作为默认大小，也可以通过设置 font-size 来控制大小。 */
.personalClass {
  font-size: 40px;
}
